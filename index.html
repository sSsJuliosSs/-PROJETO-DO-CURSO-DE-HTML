O QUE SÃO CLIENTS?

(VOLTAR)

Client, traduzido para português, clientes, é o portal de informações da internet, hoje sabemos que a internet é uma grande rede de computadores interligados, onde temos os clientes que são o meio pelo qual se buscam as informações que estão no servidor (lugar onde se armazenam essas informações).

Clientes são sites usados para buscar informações da internet, é uma maneira de acessar as informações contidas dentro de um servidor.

O QUE SÃO SERVER?

(VOLTAR)

Server, traduzindo para português, servidor, é o local onde são armazenados as informações e banco de dados, assim disponibilizando essas informações para onde forem necessários.

Os servidores podem ser local, para distribuir informações no mesmo ambiente e internas, como pode ser servidores externos, para distribuir informações para internet, ancorar informações para sites.

O QUE É PROGRAMAÇÃO?

(VOLTAR)

Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas. Ou seja, um conjunto de informações que utiliza regras para que o computador entendas o que foi solicitado. Essa linguagem de programação foi criada para padronizar no mundo inteiro as informações para que todos possam trabalhar da mesma maneira e utilizar as mesmas informações.

Nos temos algumas linguagens de programação que rodam do lado do servidor e algumas que rodam ao lado do cliente.

No servidor nos temos algumas linguagens mais conhecidas como PHP, Ruby, Csharp, Java script entre outras. Elas são direcionadas para servidores, algumas podendo serem usadas em linguagens clientes também.

Já no cliente são utilizados linguagens com HTML e CSS que não são considerados linguagens de programação, utilizamos o Java script, essa sim é uma linguagem de programação também para clientes.

QUE É HTML?

(VOLTAR)

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.
